# PeekAPI Community

Bug reports, feature requests, and discussions for [PeekAPI](https://peekapi.dev) — plug-in analytics for any REST API.

## How to get help

- **Bug reports** — [Open an issue](https://github.com/peekapi-dev/community/issues/new?template=bug_report.yml)
- **Feature requests** — [Open an issue](https://github.com/peekapi-dev/community/issues/new?template=feature_request.yml)
- **Questions & discussions** — [Start a discussion](https://github.com/peekapi-dev/community/discussions)
